# CurlHttpClient
a libcurl c++ warpper for http

# 概述
怀旧操作。
该库简单易懂易用，最重要的是一个非常不错的c++&cmake实践和参考模板
本库可以理论上支持win/lin编译，由于偷懒cmakelists中没有进行linux的配置。
